## Functionalities

-   [x] Build React App without interacting with the CLI
-   [x] Live Hot Reload

## INSTALLATION

```
git clone https://github.com/gulraiznoorbari/React-Webpack-Boilerplate.git
cd React-webpack-boilerplate
npm install
npm start
```

## For Production

```
npm build
```
